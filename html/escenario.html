ESCENARIOS

Los escenarios son parte básica de algunos métodos de análisis orientados a objetos y son representaciones en las cuales el usuario interactúa con el sistema, este esquema representa gráficamente las salidas, entradas, flujo de datos y el comportamiento del sistema. Los escenarios se pueden agregar detalles a alguna acción mediante el diseño de ejemplos interacción entre el usuario y el sistema.
